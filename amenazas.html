Amenazas en la web y cómo prevenirlas

El malware es software malicioso diseñado para dañar, interrumpir o tomar el control de sistemas informáticos, redes y dispositivos sin el consentimiento del usuario. Los ciberdelincuentes lo usan para robar información confidencial, como credenciales bancarias o datos personales, obtener acceso no autorizado, extorsionar con rescates o interrumpir el funcionamiento normal de los dispositivos.

Tipos de malware

Virus

Software malicioso que se “engancha” a otros programas o archivos, de modo que cuando esos programas se ejecutan, el virus también. Puede dañar datos, borrar archivos, etc.

Gusanos

Malware que se reproduce por sí mismo y se propaga a otros sistemas, a veces sin intervención del usuario. Infecta redes, dispositivos, etc. No necesita un programa huésped.

Troyanos

Se disfrazan como software legítimo para engañar al usuario y que lo instale. Después, permiten al atacante controlar el sistema, instalar más malware, robar datos, etc.

Ransomware

Cifra los archivos o bloquea acceso al sistema, y exige un rescate (a menudo en criptomonedas) para devolver el acceso.

Spyware

Malware cuyo propósito es espionaje: recopilar información del usuario sin que lo sepa, por ejemplo, contraseñas, historial, etc.

Adware

Genera anuncios no deseados; en algunos casos exhibe publicidad molesta, puede ralentizar el dispositivo y puede llevar a instalar otro malware.

¿Qué hacer si tu PC tiene malware?

Si crees que tu ordenador contiene software malicioso, puedes seguir los pasos a continuación para asegurarte de erradicar la amenaza y proteger tus datos.

1. Desconéctalo de Internet

Para proteger tus datos en línea, es una buena práctica desconectar el equipo de la red lo antes posible. Para ello, dirígete al ícono de Wi-Fi y desconéctate de Internet. Si estás usando Ethernet, puedes desconectar el cable Ethernet, o bien, desactivar temporalmente el adaptador de red.


2. Poner el PC (Windows) en Modo Seguro

Iniciar el equipo en modo seguro puede ayudarte a identificar el problema sin poner en riesgo tus datos, ya que el equipo arrancará solo con los controladores esenciales. Para activar el modo seguro, mantén presionada la tecla Shift mientras haces clic en el botón "Reiniciar" del menú de inicio. Luego selecciona Solucionar problemas > Opciones avanzadas > Configuración de inicio > Reiniciar. Después de reiniciar el equipo, presiona la tecla 4 o F4 para iniciar en Modo Seguro.


3. Realizar un escaneo completo con el antivirus

Abre tu programa antivirus y realiza un escaneo completo del sistema para detectar posibles amenazas. Si no tienes un antivirus instalado, puedes utilizar la herramienta de seguridad de Windows, llamada Windows Defender, que también ofrece protección efectiva.


4. Eliminar archivos temporales y limpiar el sistema

Eliminar archivos temporales puede ayudar a eliminar posibles archivos maliciosos ocultos. Puedes hacerlo utilizando la herramienta Liberador de espacio en disco de Windows o programas de limpieza como CCleaner.


5. Revisar procesos y elementos de inicio automáticos sospechosos

Revisa los procesos en ejecución mediante el Administrador de tareas (Ctrl + Shift + Esc). Desconfía de procesos desconocidos o que consumen muchos recursos. También revisa los programas que se inician automáticamente a través de MSConfig o Administrador de tareas (pestaña de inicio) y desactiva cualquier elemento sospechoso.


6. Limpiar navegadores y extensiones

Verifica las extensiones de tus navegadores (Chrome, Edge, Firefox) y elimina cualquier extensión desconocida o sospechosa. También, borra el caché y las cookies para evitar que el malware se mantenga en el navegador.


7. Restaurar el sistema

Si el malware persiste y no puedes eliminarlo, intenta restaurar tu sistema a un punto anterior donde el equipo estaba funcionando correctamente. Para hacerlo, busca la opción Restaurar sistema en el menú de inicio de Windows.


8. Reinstalación limpia del sistema (último recurso)

Si los métodos anteriores no eliminan el malware, una reinstalación limpia de Windows puede ser necesaria. Asegúrate de hacer una copia de seguridad de tus archivos antes de proceder, ya que este proceso eliminará todos los datos en el disco duro.


9. Cambiar contraseñas y verificar cuentas

Después de eliminar el malware, cambia las contraseñas de todas tus cuentas, especialmente aquellas bancarias y de correo electrónico. Asegúrate también de verificar que no se hayan hecho cambios en tus cuentas o que no se haya producido un acceso no autorizado.


10. Actualizar el sistema y software

Finalmente, asegúrate de tener el sistema operativo y todo tu software actualizado. Las actualizaciones frecuentes son esenciales para corregir vulnerabilidades que los atacantes podrían explotar. Para esto, ve a Configuración > Actualización y seguridad > Windows Update.
