Amenazas en la web y cómo prevenirlas

El malware es software malicioso diseñado para dañar, interrumpir o tomar el control de sistemas informáticos, redes y dispositivos sin el consentimiento del usuario.

Tipos de malware

Virus

Software malicioso que se “engancha” a otros programas o archivos, de modo que cuando esos programas se ejecutan, el virus también. Puede dañar datos, borrar archivos, etc.

¿Qué hacer si tu PC tiene malware?

Pasos que puedes seguir: • Desconectarlo del Internet • Poner el PC en Modo Seguro (“Safe Mode”) • Eliminar archivos temporales • Actualizar tu antivirus y software de seguridad • Escaneo completo del sistema • Aislar lo que se detecte • Revisar procesos y elementos de inicio automáticos sospechosos • Limpiar navegadores / extensiones • Restaurar el sistema, si es posible • Reinstalación limpia del sistema como último recurso • Cambiar contraseñas y verificar cuentas • Actualizar todo el sistema
