Amenazas

El ransomware es un tipo de malware (software malicioso) cuyo objetivo es secuestrar datos o impedir el acceso a sistemas de un usuario u organización, para luego exigir un rescate económico a cambio de liberar el acceso.

Phishing

El phishing es un ciberataque donde delincuentes se hacen pasar por una entidad de confianza (un banco, una empresa o una institución) para engañar a la víctima y obtener información confidencial como contraseñas, datos bancarios o personales, o para que descargue malware.
